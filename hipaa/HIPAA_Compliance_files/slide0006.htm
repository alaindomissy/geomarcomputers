Code Sets
nCodes sets are encoding applied to patients, treatments, and other aspects.
nHere are the "medical" coding schemes :
nICD-9-CM (volume I and II): These are the diagnosis codes. The International Classification of Diseases, 9th edition, Clinical Modification codes are for most diseases, injuries, impairments, other health problems and causes of injury, disease, and impairment.
nICD-9-CM (volume III): This code set will be used for inpatient and hospital services.
nCPT4: The Current Procedural Terminology level 4 code set will be used for physician services.
nHCPCS: The HCPCS will be used for physician services and certain other healthcare services and for substances, durable medical equipment, supplies, and other items.
nCDT3: This code set will be used for Dental services.
nNDC: the National Drug Codes will be used for drugs and biologicals.
nOther: The "administrative" code set
